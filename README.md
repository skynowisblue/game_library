# LibraryWindow

**LibraryWindow** is a Python module that wraps **Pygame** to simplify game development. It provides easy-to-use methods for creating windows, handling input, playing sounds/music, loading and drawing sprites, and detecting collisions.  

---

## Features

- Create and manage a game window  
- Handle keyboard and mouse input  
- Play sound effects and music  
- Load, scale, and draw sprites  
- Detect rectangle collisions and point-in-rectangle events  
- Simple main loop management
